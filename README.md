# Innerpipe
A Youtube client for Android made using Jetpack Compose <img src="https://github.com/user-attachments/assets/d1594cf7-8af9-4c19-9bff-ef803d32cb5c" width="40px"/>

## Features
- 🚫 Ad-free
- 👎 Return YouTube Dislikes

- ◼️◻️ Dark/Light mode
